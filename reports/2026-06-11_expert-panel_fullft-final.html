📝 清晰性专家
R1: 4 → R2: 4 ↔ 持平
立场转变：原评审默认 CD-SPI 有独立贡献价值的假设被打破。FedCDD (ICLR 2025) + CFL (2020) 的先验工作使 CD-SPI 作为"方法贡献"的论点不可维持。三个矛盾交叉暴露了深层盲点——CD-SPI 非新颖 + 本身有根本缺陷 + 容量叙事无惊讶值，三个支柱同时受根本性质疑。
核心修正

CD-SPI 明确定位为"应用于联邦 PRM 场景的诊断工具"而非独立方法贡献。在 Related Work 中主动引用 FedCDD 和 CFL，声明区别在于应用领域（step-level PRM 的散度诊断）和测量约束（要求相同提取点），而非方法新颖性。

容量叙事处理

彻底放弃"容量叙事"作为核心贡献。将容量重新定位为"实验变量"——容量只是改变散度性质的操作手段，真正的发现是 CD-SPI 揭示的结构性散度与聚合有效性之间的因果关系。

CD-SPI 修复方案

解决测量不对称性（统一从 backbone 最后一层 hidden state 提取），解决排列检验零假设过弱（使用容量匹配的集中式训练模型作为 null distribution），增加函数空间交叉验证（cosine 相似度和 JS 散度）独立于 CD-SPI。

修正后命题承诺

核心贡献非 full FT 的性能优势，而是 CD-SPI 作为诊断协议能够区分联邦聚合中的信号型与噪声型散度——这一区分维度被所有相关工作所忽略。CD-SPI 的结论限于相同嵌入提取点、相同架构的条件内。

CD-SPI 缺陷应对

不否认缺陷，在论文中明确列出局限性和测量不对称性；用 logit/输出层的对齐度（如 CKA）作为交叉验证指标；排列检验仅作为辅助，主要结论依靠效应量（Cohen's d）而非 p 值。

最坏情况测试

加入"反直觉条件"测试：如果 CD-SPI 在某种配置下（如 head-only）显示与 full FT 相似的"结构化"发散（假设被证伪），这反而是最有发表价值的反直觉结果。明确文档化这种可能性作为 Plan B。

P0* — 对称化测量

从 backbone 倒数第二层抽取嵌入（同时适用于 head-only 和 full FT），消除测量不对称性混同。同时报告对称和非对称两种测量结果，让读者判断差异来源。

P1 — 架构消融控制

三种不同线性层激活函数（ReLU、GELU、Identity）构建 head，完整运行实验。如果 head-only < partial FT < full FT 相对排序在所有架构下成立，则架构敏感性被控制。

P2 — 排列检验增强

三层递进排列检验：(a) 基准层：标签置换；(b) 中间层：同容量独立训练比较；(c) 最强层：随机特征映射替换。如果 head-only 在 (b)(c) 下不再显著区别于随机基线，则 CD-SPI 的语义解释不成立。

P3-P5 — 综合验证

P3: CD-SPI vs Wasserstein 距离区分"结构化发散 vs 随机分歧"；P4: 独立聚合质量指标（FedDYN drift, SCAFFOLD 控制变分）验证因果链；P5: CD-SPI 有界性定理——最小 CD-SPI 差异产生可测聚合性能差异的数学形式化。

核心命题转向

反直觉实验

加入"反直觉条件"：找到一种设置（如极端异质性）下低容量但结构化发散的系统优于高容量但噪声发散的系统。这直接挑战"容量越高越好"的直觉。

相变假设

验证从 head-only 到 full FT 的发散模式转变是平滑缩放还是存在 sharp 容量阈值。后者（相变式转变）比平滑缩放更有惊讶价值——如果实验中确实观测到 non-trivial 的相变点。

合成校准实验

构造已知发散结构的数据，验证 CD-SPI 是否能区分噪声发散和结构化发散（精确率 + 召回率）。在 CD-SPI 之外强制加入至少一个不受测量不对称性影响的独立指标（CKA 或 Procrustes 分析）作为平行证据。

✨ 最终改进命题（综合两轮六专家意见）

命题定位

本文提出 CD-SPI（Client Divergence Signal-Noise Partition Index）诊断框架——一个结构化的两阶段统计诊断协议，由 排列检验（permutation test） 和 主成分解释方差比分析（PCA EVR） 组成。本文不主张 CD-SPI 是全新的相似度度量（余弦相似度在 FL 中已被 CFL 2020, FedCDD 2025 广泛使用），而是将 FL 中已有的余弦相似度分析方法系统化为可区分噪声型与信号型发散的统计诊断协议，并首次在 联邦 PRM（过程奖励模型） 微调场景下系统性探究容量-发散类型-聚合性能之间的三元实证联系。

核心断言框架

在 dense backbone (Pythia-1.4B/2.8B) + FedAvg 受控基线 设置下：

  • H1（诊断显著性） 低容量配置（head-only / LoRA r<256）中，跨客户端参数差异在 CD-SPI 的双重统计标准下（排列检验 p > 0.05 + 主成分 EVR < 0.4）不可与噪声区分，且该噪声型发散与 FedAvg 聚合性能无关；
  • H2（结构性条件） 全参数 FT（~1.4B-2.8B 参数）产生的跨客户端参数差异在同一框架下呈现 统计显著的结构化模式（排列检验 p < 0.05 + EVR > 0.6），且该结构化发散与下游聚合性能正相关；
  • H3（阈值假说） 存在 容量转变阈值（通过 LoRA rank 精细网格扫描定位），跨过该阈值发散类型从噪声主导转变为语义结构主导——转变性质（平滑缩放 vs. sharp 相变）本身是实证发现的一部分；
  • H4（因果干预） 散度的结构化程度与聚合性能之间存在 因果关系（通过 CD-SPI 正则化干预实验和独立聚合指标交叉验证）。

贡献声明（递减优先级）

  • 首要贡献 CD-SPI 诊断协议——首个将发散类型（噪声 vs. 结构化）系统操作化的统计框架。本文核心贡献非 "full FT 更好"（该结果本身不具惊讶性），而是 CD-SPI 能够区分联邦聚合中的信号型与噪声型散度——这一区分维度被所有相关工作（FedCDD/CFL/OvA-LP 等仅报告发散程度或做聚类）所忽略；
  • 次要贡献 关于 容量与发散结构性质之间关系 的系统实证发现——揭示容量不仅改变发散幅度，更可能改变发散的根本性质（噪声 → 结构化）；
  • 辅助贡献 联邦 PRM 场景下的 受控实验基线 和完整的发散诊断协议，为后续联邦对齐研究提供可复现的方法论工具。

方法论要点

  • 对称化测量：所有 CD-SPI 测量从 backbone 倒数第二层 hidden state 统一提取（而非 head 输出），确保 head-only 和 full FT 的测量空间一致；同时报告对称与非对称两种测量结果供读者判断差异来源；
  • 多层 null distribution：建立三层递进排列检验——(a) 客户端标签置换；(b) 容量匹配的独立训练基线（集中式训练模型的参数散布）；(c) 随机特征映射替换；
  • 交叉验证：CD-SPI 之外强制加入 CKA（Centered Kernel Alignment）和函数空间 JS 散度作为独立指标，要求 CD-SPI 结论至少被一个独立指标重申；
  • 架构消融：在三种激活函数（ReLU, GELU, Identity）下验证 CD-SPI 散度排序的一致性；
  • 效应量为主：排列检验 p 值仅作为辅助展示，主要结论依靠 Cohen's d 效应量。

证伪条件

  • (F1) 若 head 维度增至 1024 后 CD-SPI 显示结构化模式 → 当前 "head-only 必然噪声" 假设受容量限制，需修正容量阈值；
  • (F2) 若 LoRA (r=64) 同时达到 full FT 95% 性能且 CD-SPI 显示为噪声 → "结构化发散是聚合有效的必要条件" 被证伪，论文退回到 Plan B（发散程度 vs. 聚合性能）；
  • (F3) 若 head-only 与 full FT 的 CD-SPI 仅呈现定量差异（程度不同）而非定性差异（噪声 vs. 结构）→ 二分法不成立，需退化为连续谱叙事；
  • (F4) 若独立指标（CKA/JS 散度）与 CD-SPI 结论不一致 → CD-SPI 引入系统性偏差，需放弃 CD-SPI 作为主要证据；
  • (F5) 若集中式基线中 head-only 与 full FT 性能差距 ≤ 2% → 问题非联邦场景特有，联邦特异性 claim 不成立。

局限性声明

因果推断限于 dense backbone (Pythia 系列) + FedAvg 聚合 + 标准异质性设定。结论在以下条件下的外推性需独立验证：(1) 非 FedAvg 聚合方法（FedProx, SCAFFOLD, FedDYN）；(2) 不同模型架构（LLaMA, Qwen, Mistral）；(3) 极端异质性（非 IID 程度 > 实验设定）；(4) PM 而非 PRM 场景。本文不主张 full FT 是联邦 PRM 的唯一路径（必要性主张），也不声称容量-性能关系具有单调性。CD-SPI 作为诊断工具，其本身体系存在着本文披露的测量不对称性、架构敏感性等已知局限，所有结论应结合交叉验证指标综合解读。

🏃 实验调整建议（融合两轮评审 + 矛盾解决）

以下实验方案融合了第一轮和第二轮所有专家的控制实验建议（P0-P6 + P0*-P5 + 合成数据校准 + 独立指标交叉验证），按优先级排列。

P0 关键路径

集中式基线 Sanity Check

在 VersaPRM 上建立 head-only / LoRA / full FT 的 centralized 准确率上限，验证 99.7% 天花板是否存在。若差距 ≤ 2% 则联邦特异性 claim 不成立。

来源：R1 清晰性 + R2 证伪条件 F5
P0 关键路径

对称化 CD-SPI 测量（P0*）

从 backbone 倒数第二层统一提取嵌入。同时报告对称和非对称两种测量结果。如果原始不对称效应在使用对称测量后消失或反转，则核心 claim 需重新评估。

来源：R2 机制 P0* + R2 清晰性
P0 关键路径

训练前 CD-SPI 基线

第 0 轮（训练前）测量 head-only 和 full FT 的 CD-SPI，控制初始化混淆。如果训练前已显示 head-only 结构化模式，则发散来源非数据驱动。

来源：R1 机制 P0
P0 关键路径

容量连续谱核心实验

LoRA r=8/64/128/256 + partial FT + head-only + full FT。重点在 rank 精细网格扫描（r=128 附近加密），定位容量阈值转变点。

来源：R1 专家组 + R2 真实空白
P1 高优先级

架构消融控制实验

三种激活函数（ReLU, GELU, Identity）构建 head。如果在所有架构下 head-only < partial FT < full FT 的 CD-SPI 排序一致成立，则架构敏感性被控制。

来源：R2 机制 P1 + R2 清晰性
P1 高优先级

三层排列检验增强

(a) 标签置换；(b) 容量匹配独立训练基线；(c) 随机特征映射替换。如果 head-only 在 (b)(c) 下不再显著区别于随机基线，则 CD-SPI 的语义解释不成立。

来源：R2 机制 P2 + R2 清晰性
P1 高优先级

独立指标交叉验证

CD-SPI 之外强制加入 CKA（Centered Kernel Alignment）和函数空间 JS 散度（在统一 hold-out 集上计算客户端输出分布发散）。要求 CD-SPI 结论至少被一个独立指标重申。

来源：R2 清晰性 + R2 惊讶元素
P1 高优先级

合成数据校准实验

构造已知发散结构的数据（ground truth：已知噪声发散 / 结构化发散），验证 CD-SPI 的精确率 + 召回率。校准 CD-SPI 的统计阈值选择。

来源：R2 惊讶元素 + R2 真实空白
P2 标准优先级

散度指向性区分实验

CD-SPI vs Wasserstein 距离（权重分布）。如果排名高度相关 → 发散源于随机漂移；如果 CD-SPI 差异在 Wasserstein 匹配后仍存在 → 发散可能有结构性质。

来源：R2 机制 P3
P2 标准优先级

独立聚合指标验证

引入 FedDYN client drift 度量、SCAFFOLD 控制变分估计等 FL 领域公认的聚合质量指标。如果 CD-SPI 与独立指标结论不一致，则 CD-SPI 可能引入系统性偏差。

来源：R2 机制 P4
P2 标准优先级

反直觉条件测试

构造极端异质性场景，验证是否存在"低容量结构化发散优于高容量噪声发散"的反直觉情况。如果是，这是最有发表价值的发现。

来源：R2 惊讶元素 + R2 真实空白（最坏情况测试）
P2 标准优先级

CD-SPI 理论形式化

建立 CD-SPI 的有界性定理、尺度不变性证明、与 FedAvg 收敛率之间的理论联系。至少需要 Lemma 级别的形式化结果，而非停留在"高 = 好"直觉层面。

来源：R2 机制 P5 + R2 惊讶元素 + R1 全体
P2 标准优先级

OOD 鲁棒性与异质性消融

OOD 跨域鲁棒性测试 + 异质性梯度消融（验证 head-only 在高异质性下是否变为结构化）+ A100 对照实验（控制设备影响）。

来源：R1 专家组 + R2 清晰性
P2 标准优先级

CD-SPI 因果干预实验

用正则化惩罚控制 CD-SPI 值，检验 CD-SPI 的因果效应。如果干预 CD-SPI 后聚合性能随之改变，则因果链得到直接支撑。

来源：R1 机制 P4

实验策略建议

  • 先跑 P0 再理论化：当前最紧急的是集中式基线 + 对称 CD-SPI + 训练前基线 + 容量连续谱。零实验数据是惊讶专家继续下调评分的根本原因。
  • Plan B 的文档化：预先设定每条证伪条件的触发后行动。如 F3（无定性差异）触发 → 退化为连续谱叙事而非二分法。Lab Notebook 级别的决策树应提前建立。
  • 数据集对齐：容量天花板 99.7% 来自 PRM800K 而非 VersaPRM。在 VersaPRM 上重新建立 centralized 基线（head-only, LoRA r=8/64/256, full FT）以避免原始前提证据无效问题。
  • 多指标平行展示：CD-SPI、CKA、函数空间 JS 散度三者平行报告。不依赖单一指标做核心 claim。在论文中明确列出 CD-SPI 的已知局限性。
  • 迭代循环：每完成一个实验集群（P0, P1, P2）后召集一次 mini-panel 评估，避免一次性运行所有实验后才发现方向性错误。

🎯 最终总结

两轮评审的综合判定

四专家 panel 从第一轮平均 3.25/10 微幅改善至第二轮 3.50/10（+0.25）。改善主要来自真实空白和机制联系两位专家对命题调整方向的认可（各 +1），被惊讶专家因无实验数据而更严格的判定（-1）部分对冲。

主要改善

(1) 三矛盾中两项被解决（容量叙事争议、CD-SPI 缺陷可修复性），一项部分解决（CD-SPI 贡献价值——方向趋同但依赖实验数据）；
(2) 论文命题从"容量更好的规模化验证"彻底重构为"CD-SPI 区分噪声/结构化发散的分析框架"——叙事骨架显著强化；
(3) CD-SPI 的三个设计缺陷均有对应的具体实验控制方案（对称测量、多层 null、交叉验证）；
(4) 所有专家对容量叙事的定位达成一致——降级为实验变量。

剩余风险

(1) 零实验数据仍然是最根本的风险——所有命题调整和矛盾解决都建立在"假设实验结果符合预期"的前提下，而 (a) 实验结果完全可能反预期（证伪 CD-SPI 的价值），(b) 即使结果正预期，其惊讶值也需要实际数据来证实；
(2) CD-SPI 的分析空间（噪声 vs 结构化区分）被所有专家承认为"薄层"——即使完美执行实验和修复，仍不足以支撑顶级会议主贡献的可能性存在；
(3) CD-SPI 的三个缺陷虽然"可控制"，但控制实验（P0*-P5）增加了大量工作量，与 4 个月的有限时间窗口形成张力；
(4) 惊讶专家评分低至 2/10 是一个严重信号——如果实验数据不展示非平凡模式（相变、反直觉条件、非单调性），论文发表前景堪忧。

最终建议

立即开始实验。任何进一步的命题讨论在没有数据支撑的情况下边际收益递减。按 Phase 0 集群（集中式基线 + 对称 CD-SPI + 训练前基线 + 容量连续谱）启动，在 2 周内获得第一批关键数据点。建立 Lab Notebook，预先记录每条证伪条件（F1-F5）的触发后行动。理论形式化与实验并行进行，避免理论脱离数据的空中楼阁。在第 8 周执行终止锚点评估——不进入无限优化循环。
